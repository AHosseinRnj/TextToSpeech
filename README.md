# Text To Speech
>This application uses [Microsoft Speech API (SAPI)](https://docs.microsoft.com/en-us/previous-versions/windows/desktop/ms723627(v=vs.85)), which is implemented with Object Oriented paradigm.

### Features
* Voice gender
	* Male
	* Female
* Voice speed
	* Value between -30 & +20
* Voice volume
	* Value between 0 & 100
